Features

  • Electro-hydraulic gearbox features easy operation. The CPCD120 is integrated with multi-plate clutch for quick speed control,enabling stable operation.

  • Reinforced drive axle enables it to be more suitable for use under harsh working conditions.

  • A protective sleeve is added to the steering cylinder to reduce the external force damage of the cylinder.

  • Low-loss multi way valve application ensures lifting system efficiency is increased by 10%, oil temperature decreased by 12 ℃.

  • The integral overhead guard can stand high pressure, which is rainproof and sun-proof, making it suitable for outdoor operation.

  • 180L large-capacity fuel tank provides longer working time for a single refueling.

  • With counterbalance mesh cover, the engine and control system can be well-protected for safe operation.

  • Ratchet-type brake system requires less effort from the operator to set.

  • Intelligent instrument realizes live preview of vehicle operation information in a real-time manner.

  • Three-layer parallel composite radiator can raise the heat dissipation effect by 15%.

  • Combustion system is covered by gasket seal to prevent leakage, making clean and safe operation.

  • 12-ton heavy-duty structural wide-view mast
